# Agile Scrum (2-Week Sprints)
**Core Discipline:** Focus on rapid MVP iterations. Code can be pushed to feature branches if unit tests pass locally. Stand-up updates required daily.
**Protocols:**
- [Sprint Planning]: Break down User Stories into sub-tasks.
- [TDD Loop]: Write test -> Write Code -> Refactor.
- [PR Review]: Hand off to $$PM/$$REVIEWER for rapid Code Review.
