# 系統總體架構白皮書：提線社畜 (即夢旗艦版)

這座名為 **「提線社畜 (The Marionette's Requiem)」** 的全自動 B2B 影音兵工廠，採用「即夢 AI 高級會員 (Premium)」作為核心視覺引擎，並以「MiniMax T2A」確保音軌的 100% 商業版權合規。

架構分為兩個獨立運作的部門：
* **【⚙️ 研發部 (Gear 2)】**：24小時自動巡邏 YouTube/B站，萃取全世界最酷的神級美學，存入 SQLite 金庫。
* **【🏭 生產部 (Gear 1)】**：接到痛點訂單後，去金庫抽取基因，精準組裝出高品質商用微電影。

---

### 🌟 核心造車鐵律 (Core Disciplines)
1. **音樂絕對先行**：必須先呼叫 MiniMax 生成合法配樂並量測精確秒數後，才能寫分鏡劇本。
2. **視覺物理閹割 (Aesthetic Lock)**：雖然使用高階的即夢 AI，但必須在 Prompt 中強制植入「定格動畫、木偶、卡頓感」，以掩蓋 AI 瑕疵並維持厭世美學。
3. **無人聲純器樂與版權紅線**：嚴禁爬取網易雲等串流平台。配樂必須使用 MiniMax 或免版稅庫 (Pixabay)，且禁止語音對嘴 (TTS)，全靠純樂器 + 實體白噪音 (ASMR) 撐起情緒。
4. **聽覺絕對剝奪 (Audio Ban)**：即夢 AI 產生的任何影片必須強制靜音 (Mute)，僅能保留 `$$AD` (混音師) 製作的合法音軌。

---

### ⚙️ 研發部：R&D 逆向研發線 (無限永動機)
**目標：全自動把神級美學塞入 SQLite 基因庫 (`style_vault.db`)。**

1. **AI 藝術總監發想** (`infinite_miner_daemon.py` -> `daily_keyword_mutator.py`)：呼叫智譜 GLM-4-Plus，把「極端美學」和「社畜情境」隨機碰撞，產生冷門搜尋關鍵字。
2. **多模態礦車出動** (`batch_style_analyzer_sqlite.py`)：
   * 去 YouTube / B 站搜尋並下載影片。
   * 自動切出 12 張截圖，並**「拼接成一張十二宮格縮圖表 (Contact Sheet)」**以繞過 API 限制。
   * 丟給視覺大腦 `glm-4v-plus` 提煉「視覺基因」與「聽覺基因」，存入 `style_vault.db`。
3. **系統休眠保護**：休眠 4 小時防封鎖，隨後無限循環。

---

### 🏭 生產部：日常自動化產線 (The Assembly Line)
**目標：將痛點與基因縫合，全自動產出 3 分鐘高品質商用短劇。**

* **步驟 1：抓取痛點 (SDR)**
  `SDR` 爬梳論壇痛點交給 `PM`，並透過 Telegram 傳給老闆。老闆點擊 **【✅ 批准】** 後啟動產線。
* **步驟 2：合成獸企劃藍圖 (PM + CW)**
  `PM` 從 SQLite 抽出 **3 種不同基因**。`CW` 呼叫 GLM-4-Plus，將痛點與 3 種基因融合，產出**「企劃藍圖 JSON」** (包含即夢專用中文 Prompt 與 MiniMax 專用英文配樂 Prompt)。
* **步驟 3：音樂生成與測量 (AD) - [Music First]**
  `AD` 拿著藍圖呼叫 MiniMax T2A 算音樂，疊加 ASMR，並測量出**精確長度（如：168 秒）**，回報給 `PM`。
* **步驟 4：精準分鏡切割 (CW)**
  `CW` 根據 168 秒限制，寫出對齊節奏的分鏡劇本 (`storyboard.json`)。`PM` 將音樂與劇本傳至 Telegram 進行 **【✅ 老闆審核】**。
* **步驟 5：視覺旗艦算圖 (VD)**
  `VD` 帶著即夢專屬 Prompt 呼叫 Jimeng AI Premium API。強制附加「定格動畫/木偶」關鍵字，並徹底捨棄即夢生成的音軌。
* **步驟 6：組裝與版權護身符 (QA)**
  `QA` 執行 `auto_editor.py`，將無聲影像與 168 秒合法配樂無縫對齊，壓上字幕。同時生成 `receipt.json` (版權來源收據)。
* **步驟 7：核算與發布 (PM)**
  `PM` 傳送最終 .mp4 至 Telegram 進行 **【✅ 老闆最終確認】**。批准後自動發布、記帳、打包 ZIP 傳至 S3 冷儲存並清空本機暫存。

### 🛡️ 系統安全氣囊 (Exception Protocols)
1. **版權與內容違規停損**：若遇 API 內容審查退件，記錄沉沒成本；若月耗損 > $5，自動修改爬蟲濾網。
2. **破產與限流警報**：遭遇 402/429 錯誤時，立刻切斷產線並發送 Telegram 警報，防止帳號被封。
3. **營收進化迴圈**：每週分析「前 48 小時完播率」，將最受歡迎的「基因+痛點」回寫入學習日誌，引導未來藍圖生成。
