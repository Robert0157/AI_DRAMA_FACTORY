### 🌟 核心造車鐵律 (Core Disciplines)
在工廠開工前，所有 AI 員工都必須遵守的底層規則：
1. **音樂絕對先行**：必須先有音樂，有了確切的音樂秒數，才能寫分鏡劇本。
2. **擁抱 AI 瑕疵**：全部採用「定格動畫/微縮模型/木偶」風格，掩蓋 AI 算圖不連貫的物理缺陷。
3. **無人聲純器樂**：禁止 AI 語音對嘴 (TTS)，全靠純樂器配樂 + 實體白噪音 (ASMR) 撐起情緒，打破語言隔閡。
4. **絕對禁止政治**：只專注於全球共鳴的「社畜/職場痛點」。

### ⚙️ 研發部：R&D 逆向研發線 (無限永動機)
**目標：趁著你的 2000 萬 Token 沒過期，全自動把全世界的神級美學塞進你的 SQLite 基因庫 (`style_vault.db`)。**

這是一個 24 小時在 Windows 電腦上背景運作的「無限迴圈 (`infinite_miner_daemon.py`)」：

1. **AI 藝術總監發想** (`daily_keyword_mutator.py`)：呼叫智譜旗艦大腦，把「賽博龐克/吉卜力/侘寂」和「加班/開會」隨機碰撞，產生 30 個極度冷門的搜尋關鍵字（寫入 `video_input.txt`）。
2. **多模態礦車出動** (`batch_style_analyzer_sqlite.py`)：
   * 讀取關鍵字，去 YouTube / B 站搜尋並下載前 3 分鐘影片。
   * 自動切出 12 張截圖。
   * 把截圖丟給視覺大腦 `glm-4v-plus`，請它「反向推導」出這支影片的**「視覺基因」**與**「聽覺基因」**。
   * 提煉完畢後，存入 `style_vault.db` 金庫。
3. **系統休眠**：睡 4 個小時防封鎖，醒來後繼續重複步驟 1。

---

### 🏭 生產部：日常自動化產線 (The Assembly Line)
**目標：將痛點與基因完美縫合，全自動產出 3 分鐘高品質短劇。**

當老闆（你）決定要開拍一部短劇時，OpenClaw 多智能體框架會依照以下順序精準交接：

#### 步驟 1：抓取痛點 (SDR)
* `SDR` 去 Reddit/Dcard 爬梳出 10 個職場痛點，傳給 `PM`。
* `PM` 將這 10 個痛點傳到 Telegram。
* 🛑 **【防線 1：老闆選題】** 你在手機上挑選一個最有感的痛點，並批准執行。

#### 步驟 2：合成獸企劃藍圖 (PM + CW)
* `PM` 轉身打開 SQLite 基因庫，隨機抽出 **3 種完全不同的基因**（例如：蘇聯粗野主義 + 黏土動畫 + 迷幻電子樂）。
* `CW` (編劇) 擔任煉金術師，將「你的痛點」與「這 3 種基因」強行縫合，寫出一份**「企劃藍圖 (Blueprint)」**。
* 這份藍圖包含：全新的融合視覺 Prompt、融合聽覺 Prompt、以及三國語言的故事大綱（此時還沒有確切的分鏡數）。

#### 步驟 3：音樂生成與測量 (AD)
* `PM` 把藍圖裡的「融合聽覺 Prompt」交給 `AD` (音效師)。
* `AD` 去呼叫 Suno/Udio 算音樂，並疊加鍵盤聲、時鐘聲等 ASMR。
* `AD` 測量出這首音樂的**精確長度（例如：168 秒）**，回報給 `PM`。

#### 步驟 4：精準分鏡切割 (CW)
* `PM` 把「藍圖 + 168 秒」交還給 `CW` (編劇)。
* `CW` 根據 168 秒的限制，精準寫出完美對齊節奏的 JSON 分鏡劇本 (`storyboard.json`)。
* 🛑 **【防線 2：老闆審查劇本與音樂】** `PM` 將 168 秒的音樂與 JSON 劇本傳到 Telegram。你聽過、看過覺得滿意，按下批准，系統才會進入「最花錢」的算圖階段。

#### 步驟 5：視覺算圖 (VD)
* `VD` (視覺導演) 拿到劇本，帶著那組「融合視覺 Prompt」作為基底，去呼叫 Kling/Midjourney，把每一幕的定格動畫算出來，存入資料夾。

#### 步驟 6：組裝與版權護身符 (QA)
* `QA` (剪輯師) 執行 Python 腳本，把影像和 168 秒的音樂無縫對齊，壓上三國語言字幕。
* 同時生成一份 `receipt.json`，記錄這支影片用了什麼 AI 引擎、什麼音效來源，確保未來商用的版權安全。

#### 步驟 7：核算與發布 (PM)
* 🛑 **【防線 3：老闆最終確認】** `PM` 把剪好的 .mp4 傳到 Telegram 給你做最後確認。
* 你批准後，系統自動上傳 YouTube / Spotify。
* `PM` 將這次花費的 API 成本寫入記帳資料庫。
* 系統將所有暫存檔打包成 ZIP 送上 S3 雲端冷儲存，清空本機硬碟。

### 🛡️ 系統的「安全氣囊」 (異常與進化機制)
為了確保這座工廠不會因為意外而崩潰破產，我們安裝了三個底層防護：

1. **內容違規停損**：如果算圖時被 API 原廠判定「色情/暴力」退件，PM 會記錄這筆「沉沒成本」。如果這個月因此浪費超過 $5 美金，系統會自動修改 SDR 的爬蟲濾網，以後不再抓這種危險痛點。
2. **破產警報**：如果 API 餘額不足或遇到大當機 (HTTP 402/429)，系統會立刻切斷產線，傳 Telegram 叫你來處理，防止帳號被原廠封鎖。
3. **營收進化迴圈**：每個禮拜，SDR 會去抓 YouTube 的「前 48 小時完播率」。找出最受歡迎的「基因 + 痛點」組合，寫入學習日誌，讓 AI 知道以後要多配對這種風格。
