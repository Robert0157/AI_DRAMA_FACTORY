# 系統總體架構白皮書：提線社畜 (游擊隊 RPA 旗艦版 v7.0)
**發布日期**: 2026-03-25 | **核心戰略**: 零 API 成本、無頭瀏覽器自動化、DOM 節點即時監控
**目標環境**: Mac mini M4 (Apple Silicon) + 外部 APFS 硬碟 (`/Volumes/AI_Workspace/`)

## ▌ 0. 系統總體哲學與邊界限制
本系統為極限壓縮成本的 B2B 商業微電影產線。視覺生成**全面棄用官方 API**，改採 Playwright 無頭瀏覽器 (Headless Browser) 操控即夢 (Jimeng) 高級網頁版。
* **低成本鐵律**：視覺生成預算鎖死於每月即夢 Premium 訂閱費，不按件計費。
* **DOM 監控鐵律**：因依賴網頁前端結構，必須部署「前端哨兵」即時監控網頁改版，做到「發現改版 -> 立即停機 -> 人工修復」，嚴禁產線盲目執行導致死循環或帳號異常。
* **音樂先行與瑕疵美學**：維持 `MiniMax T2A` 產生音軌，並強制植入「定格動畫」等關鍵字。

## ▌ 1. 全新防禦陣線：前端哨兵系統 (The DOM Sentinel)
**目標：即時探測即夢網頁改版，保護高級帳號不被反爬蟲機制封鎖。**
* **新增模組**：`jimeng_dom_sentinel.py`
* **運作機制 (Pre-flight Check)**：
  1. **心跳巡邏**：設定為每 2 小時背景執行一次，以及在 `$$VD` (視覺導演) 正式啟動前 **強制執行一次**。
  2. **特徵碼比對 (Hash & Selector Check)**：哨兵會發送輕量級 HTTP 請求，檢查即夢生成頁面的關鍵 XPath 與 CSS Selectors（如：`textarea.prompt-input`, `button.generate-btn`, `.video-download-icon`）是否存在。
  3. **異常攔截**：若發現 DOM 結構改變（找不到按鈕或特徵碼不符），哨兵會立刻回傳 `Exit Code 1`。
  4. **警報通報**：觸發 `telegram_approver_audio.py` 的警報廣播：「🚨 警告：即夢前端已改版！產線已強制暫停，請 RD 團隊立即更新 XPath 選擇器！」

## ▌ 3. 齒輪一：日常自動化產線 (Gear 1 - OpenClaw RPA 版)

### 3.1 [Protocol PM-Init] 合成獸藍圖工廠 (`batch_chimera_blueprint_factory.py`)
* `$$PM` 抽出 3 筆基因，交由 `$$CW` 縫合產出 `blueprint_xxx.json`。

### 3.2 [Protocol A] 音效套利與測量 (`chinese_api_audio_engine.py`)
* 呼叫低價 API 生成純器樂配樂，經 `ffprobe` 測量秒數後交給 `$$CW` 切割分鏡。

### 3.3 [Protocol S] Telegram 審批 (`telegram_approver_audio.py`)
* CEO 在 Telegram 上聽完音樂、看完劇本摘要後，點擊【✅ 批准】放行。

### 3.4 [Protocol V] RPA 游擊算圖引擎 (`jimeng_rpa_engine.py`) 🚀 **【全新核心】**
* **淘汰舊模組**：移除所有即夢 API 呼叫腳本。
* **新模組邏輯**：
  1. **哨兵放行**：確認 `jimeng_dom_sentinel.py` 回報安全。
  2. **Playwright 啟動**：在 Mac M4 背景啟動 Chromium，載入已登入即夢 Premium 的 Browser Context (Cookie / Session)。
  3. **擬人化輸入**：讀取 `storyboard.json` 與基準圖 (`image_anchors`)。使用 `page.type(delay=100)` 模擬人類打字輸入 Prompt，並模擬滑鼠軌跡點擊生成。
  4. **非同步等待**：監控畫面上的進度條 DOM，直到出現 `video-ready` 標籤，執行網頁原生下載，並將 .mp4 存入 `/Volumes/AI_Workspace/AI_DRAMA_FACTORY/assets/video_clips/`。
* **防封鎖機制**：每次生成後強制加入 `random.uniform(30, 90)` 秒的隨機擬人化休眠。

### 3.5 [Protocol Q] M4 硬體加速剪輯台 (`auto_editor.py`)
* 使用 Apple Silicon `h264_videotoolbox` 將 RPA 下載的無聲影像與配樂合併，壓製字幕，產出 `receipt.json`。

### 3.6 [Protocol D] 財務核算與垃圾回收 (`ledger_manager.py` / `cloud_archiver.py`)
* 記帳 (此時視覺成本 Token = $0)，打包歸檔至 S3。
